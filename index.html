Unofficial Tango Controls package repository

This repository contains compiled packages for TANGO Controls: https://sourceforge.net/projects/tango-cs/

Debian packages

The packages listed below have no dependencies declared. Make sure you have following packages in your system before proceeding: libmysqlclient20, libomniorb4-1, libzmq5, libcos4-1.

The only supported system is Ubuntu 16.04!

APT repository

Configure APT as shown below:


echo "deb [trusted=yes] http://mliszcz.github.io/tango-cs-build/repository/ apt/" >> /etc/apt/sources.list
apt-get update
